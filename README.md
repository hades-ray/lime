# lime

Это проект моей курсовой работы, здесь представлен сайт маркетплейса с готовой версткой и БД

## Требования
- XAMPP (или аналогичный стек: Apache, PHP 7.4+, MySQL 5.7+)
- Браузер

## Установка на XAMPP

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/hades-ray/lime.git

2. **Разместите проект в XAMPP:**
   
- Скопируйте папку проекта в C:\xampp\htdocs\ (или ваш путь к XAMPP)

- Переименуйте папку, если нужно

3. **Настройте базу данных:**

- Откройте phpMyAdmin: http://localhost/phpmyadmin

- Создайте новую базу данных (например, myproject_db)

- Нажмите на созданную БД → вкладка "Импорт"

- Выберите файл database/user_structure.sql из проекта

- Нажмите "Выполнить"

4. **Настройте конфигурацию:**

- Скопируйте config.php.example в config.php

- Откройте config.php и измените настройки:

- $db_pass = ''; // Пароль от MySQL
- $db_name = 'lime'; // Название вашей БД

5. **Запустите проект:**

- Откройте в браузере:
  ```
  https:/localhost/ваша_папка/index.php

- Для входа зарегистрируйте тестового пользователя
